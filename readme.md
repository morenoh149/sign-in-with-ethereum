# Sign with Ethereum

login with web3 wallet

Demo at https://harrymoreno.com/sign-in-with-ethereum/ (served from github).

## Running

* serve locally `python3 -m http.server 8000 --bind 127.0.0.1`
* open `http://localhost:8000`
