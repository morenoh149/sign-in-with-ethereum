# Sign with Ethereum

login with web3 wallet

![screen](gif.gif)

Demo at https://sign-in-with-ethereum.netlify.app/.

## Running

* serve locally `python3 -m http.server 8000 --bind 127.0.0.1`
* open `http://localhost:8000`

## Dependencies

* https://docs.ethers.io/v5/
